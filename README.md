# Personal Portfolio Website

The Personal Portfolio Website is a showcase of my skills, projects, and experiences, designed to provide an overview of my current professional background and accomplishments.


<h2>Features-</h2>
<li>Display personal information, including name, bio, and contact details</li>
<li>Showcase projects, work experience, education, and skills</li>
<li>Interactive design for seamless navigation and engagement</li>
<li>Responsive layout for optimal viewing on various devices</li>


<h2>Technologies Used-</h2>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>


<h2>Usage-</h2>
<li>Navigate through different sections to explore the portfolio content</li>
<li>View projects, work experience, education, and skills with detailed information</li>
<li>Contact the owner using the provided contact details or contact form</li>
<li>Customize and update the portfolio content as needed</li>
